# Home Assistant App: Zabbix Proxy

_Zabbix proxy with SQLite3 storage for Home Assistant._

![Supports aarch64 Architecture][aarch64-shield]
![Supports amd64 Architecture][amd64-shield]

This app runs the official `zabbix/zabbix-proxy-sqlite3:alpine-7.4-latest`
image and stores the proxy SQLite database in Home Assistant app data.
